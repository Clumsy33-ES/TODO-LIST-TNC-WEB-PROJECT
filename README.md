# React Todo List Projesi

Bu proje React kullanılarak geliştirilmiş basit bir Todo List (Yapılacaklar Listesi) uygulamasıdır.

Proje, React temel konularını pekiştirmek amacıyla yapılmıştır.

---

## Özellikler

- Yeni görev ekleme
- Görev silme
- Görevi tamamlandı olarak işaretleme
- Görevi önemli olarak işaretleme
- Görevleri 3 ayrı sütunda görüntüleme:
  - Important (Önemli)
  - Completed (Tamamlanan)
  - All Tasks (Tüm Görevler)

---

## Kullanılan Teknolojiler

- React
- JavaScript (ES6)
- CSS
- Vite
